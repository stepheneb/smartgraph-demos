Ball Drop using scripty2: S2.FX.Transitions.bounce(pos)

I extracted, simplified, and modified the S2.FX.Transitions.bounce(pos) demo from Thomas Fuch's documentation for S2 here: scripty2 API documentation S2.FX.Transitions in order to make a graph that shows the position of a ball dropping and bouncing.

I wanted a simpler example to more easily understand how his interactive graphing works using just css. The interaction of the graph and the ball are made without using Canvas, SVG, VML, images, etc -- only CSS. I've taken much of the code in the original html, css and in the javascript file application.js out to leave just the code needed to run this demo.
